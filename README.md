# Gopher Avatars Generator

A fun project for creating custom Go gopher avatars with a simple visual editor.

## Project Structure

- `gopher-customizer/` - React + TypeScript web app for avatar customization

## Features

- Change gopher body color with palette controls
- Select accessories (glasses, headwear, neckwear)
- Fine-tuned accessory placement support
- Export avatar as PNG
